## ✔ 주요 기능

- 🏬 가맹점관리
  - 가맹점 등록
  - 가맹점 경고 기능
  - 폐점한 매장 업데이트 및 조회

- 🍽 메뉴관리
  - 메뉴 등록
  - 메뉴 판매 상태 변경 및 조회(판매/미판매)

- 💻 주문관리
  - 가맹점 POS서버 발주 정보 API통신

- 📈 매출관리
  - 가맹점주 POS 내 매출 정보를 연월별로 API통신
  - 최저/최고 매출액 구분 및 미제공 가맹점 목록 조회

- 🗓 캘린더관리
  - 부서별 캘린더 조회 및 등록
  - 캘린더별 일정 관리

- 📋 전자결재
  - 결재 양식을 선택하여 기안문 상신
  - 결재자 수에 따른 결재라인 생성
  - 기안문 승인 및 반려
  - 결재 순서에 따른 메뉴 구분
  - 전자결재 문서 PDF 및 인쇄 기능
